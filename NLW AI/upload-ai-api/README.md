# upload.ai API

## Technologies used:
- OpenAI Whisper
- OpenAI ChatGPT API
- TypeScript
- Fastify
- Prisma
- SQLite

## Instalation

To install back-end dependencies, run:
```sh
pnpm install
```

To apply database migrations, run:
```sh
pnpm migrate dev
```

To browse database data, run:
```sh
pnpm prisma studio
```

pnpm prisma db seed



## Running

To run the back-end, use:

```sh
npm run dev
```
